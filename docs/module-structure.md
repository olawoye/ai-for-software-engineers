# Module and Lesson Mapping

Use this document to trace each `module-XX` folder back to the course curriculum.

1. **module-01-ai-shift-for-engineers**: Covers "AI Shift For Engineers" lessons (mindset, patterns, transition plan).
2. **module-02-ai-fundamentals**: Implements the foundational LLM concepts and the mini search demo.
3. **module-03-ai-developer-toolkit**: API calling, rapid prototyping, chat interfaces, DevOps, and mini deployments.
4. **module-04-practical-rag-context-engineering**: Vector stores, embeddings, pipeline construction, and RAG optimizations.
5. **module-05-developing-mcp-servers-tooling**: MCP server architecture, tooling, and observability patterns (placeholders for future lessons).
6. **module-06-ai-agents-autonomy**: Agent design, tool use, multi-step action planning, and orchestration.
7. **module-07-designing-ai-native-systems**: Systems thinking, UX, feedback loops, and AI-native design principles.
8. **module-08-production-ai-systems**: Production-grade monitoring, scalability, data validation, and governance.
9. **module-09-advanced-capabilities-specializations**: Advanced topics such as ML, mechanistic interpretability, etc.
10. **module-10-career-transition-monetization**: Research, career narrative, monetization strategies, and consulting preparations.

Each folder contains both a `project-completed/` lesson and a `project-todo/` scaffold so learners can either read the finished solution or build it interactively.
