# 02 VS Code Setup

1. Install the Python extension and select the `.venv` interpreter via `Python: Select Interpreter`.
2. Enable format-on-save (Black) and linting (Pylint/Flake8) for consistent results between agents and manual coding.
3. Use the integrated terminal to run `uv run` commands so the environment aligns with the script wrappers.
4. Open `project-todo/` lessons next to `project-completed/` references to keep the build path visible while coding.
5. Use the Command Palette to launch `scripts/run_streamlit.sh` or `scripts/run_cli_agent.sh` if you prefer the helper wrappers.
